# SIMPL Interpreter
SIMPL stands for Simply Interpreted Mathematical Programming Language.A hobby interpretted programming language developed by me.

# Features
- lexer
- Tokenizer
- print statements
- if statements
- variables

# How to Run
1. clone repository
2. run `make` or the command inside the makefile
3. run `simpl.exe test.txt` for standard test or alternatively supply your own text file
